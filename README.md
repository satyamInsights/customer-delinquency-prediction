# 📊 Customer Delinquency Prediction using Machine Learning

This project focuses on predicting whether a customer is likely to have a delinquent account using Machine Learning techniques. It helps in identifying high-risk customers based on financial attributes.

---

## 🚀 Project Overview

- Built a predictive model using **Random Forest Classifier**
- Analyzes customer financial data to detect risk patterns
- Supports **data-driven decision-making** in financial services

---

## 📂 Dataset Features

The model uses the following key features:

- 💰 Income  
- 📊 Credit Score  
- 💳 Credit Utilization  

Target Variable:
- ⚠️ Delinquent Account (Yes/No)

---

## 🛠️ Tech Stack

- Python 🐍  
- Pandas 🧮  
- Scikit-learn 🤖  
- OpenPyXL (for Excel handling)

---

## ⚙️ Workflow

1. Data Loading from Excel file  
2. Data Cleaning (handling missing values)  
3. Feature Selection  
4. Train-Test Split  
5. Model Training using Random Forest  
6. Model Evaluation (Accuracy Score)

---

## 📈 Model Output

- Predicts whether a customer is **delinquent or not**
- Provides insights for **risk assessment**
- Helps businesses take preventive actions
